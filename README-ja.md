## 特徴

- [x] マテリアルデザイン3（You）
- [x] ダークとライトのテーマ
- [X] 音声および画面録画のサポート
- [X] 利用可能な様々なフォーマットとコーデック
- [X] アプリ内録音プレーヤー
- [X] SAF（ストレージアクセスフレームワーク）のサポート
- [X] Jetpack Composeで書かれています
